# Images Folder

Store all your website images here.

## Recommended Images

### Project Images
- `project-1.jpg`, `project-2.jpg`, `project-3.jpg` - Main project thumbnails
- Recommended size: 800x600px or similar aspect ratio

### Blog Images
- `blog-1.jpg`, `blog-2.jpg`, etc. - Blog post header images
- Recommended size: 1200x630px for good display

### Profile Image
- `profile.jpg` - Your profile/about section image
- Recommended size: 400x400px (square) or 600x400px

### Placeholder Examples
You can temporarily use these formats in your file names:
- `project-placeholder-1.jpg`
- `blog-placeholder-1.jpg`
- `profile-placeholder.jpg`

## Image Formats
Supported formats:
- `.jpg` / `.jpeg` - Best for photos
- `.png` - Best for graphics with transparency
- `.gif` - For simple animations
- `.webp` - Modern format with better compression

## Optimization Tips
- Compress images before uploading to improve load times
- Use appropriate dimensions (don't use 4000px images if displaying at 800px)
- Consider using WebP format for better performance
